# SIJING OS · 使用说明

一个部署在 GitHub Pages 上的艺术家个人网站。
前台：极简三语（中/日/EN）展示；后台：`admin/upload.html`，上传作品、编辑履历和所有文字，网站自动更新，永远不用碰代码。

---

## 一、首次部署（只做一次，约 15 分钟）

### 1. 建仓库
1. 登录 github.com → 右上角 **+** → **New repository**
2. 仓库名随意，例如 `sijing-site`；选 **Public**；点 **Create repository**

### 2. 上传网站文件
1. 在新仓库页面点 **uploading an existing file**
2. 把本压缩包解压后的**全部内容**（index.html、style.css、script.js、assets、works、data、admin 这些文件和文件夹）一起拖进去
3. 点 **Commit changes**

### 3. 开启 GitHub Pages
1. 仓库页面 → **Settings** → 左侧 **Pages**
2. **Source** 选 `Deploy from a branch`，Branch 选 `main` / `(root)`，点 **Save**
3. 等 1–2 分钟，页面上方会出现你的网址：
   `https://你的用户名.github.io/仓库名/`

### 4. 生成访问令牌（后台的钥匙）
1. GitHub 右上角头像 → **Settings** → 最下方 **Developer settings**
2. **Personal access tokens → Fine-grained tokens → Generate new token**
3. 填一个名字（如 `sijing-admin`），有效期选 1 年（到期后重新生成一个即可）
4. **Repository access** 选 `Only select repositories`，只勾选刚才建的这一个仓库
5. **Permissions → Repository permissions → Contents** 选 `Read and write`，其余保持默认
6. 点 **Generate token**，复制 `github_pat_` 开头的字符串（只显示这一次）

### 5. 配置后台
1. 打开 `https://你的用户名.github.io/仓库名/admin/upload.html`
2. 进入「设置」标签，填入：GitHub 用户名、仓库名、分支（main）、刚才的令牌
3. 点 **保存设置** → 右上角显示「已连接」即成功

令牌只存在这台设备的浏览器里，不会上传到任何地方。换设备使用后台时，把同一个令牌再粘贴一次即可。

## 二、日常使用（以后你只需要做这些）

### 新增作品
打开后台 → 「作品」→ 选图片 → 填标题/年份/尺寸/材料/三语介绍 → 选「已售/在售」「公开/草稿」→ **保存**。
图片会自动压缩上传，网站几分钟内自动更新，按年份自动分组排序。

### 编辑 / 删除作品
「作品」标签下方的「已有作品」列表里点 **编辑** 或 **删除**。

### 新增展览 / 获奖（CV）
「履历 CV」→ 填名称、地点、年份、类别 → **保存**。CV 页自动按年份排列。

### 修改首页 / About / Statement / Research / Contact
「文字页面」→ 选对应页面 → 在富文本框里直接改（支持加粗、斜体、小标题）→ **保存**。

保存后网站更新有 1–3 分钟延迟（GitHub Pages 部署时间），刷新时可按 Ctrl/Cmd+Shift+R 强制刷新。

## 三、备份与安全

- 你的全部内容就在 GitHub 仓库里（data/ 里的三个 JSON + works/ 里的图片），仓库本身就是备份。
- 令牌 = 修改这个仓库的钥匙：不要截图外传；在公用电脑用过后台后，点「设置」里的 **清除本机设置**。
- 令牌到期或泄露：去 GitHub 撤销旧的、生成新的，重新粘贴到后台即可。

## 四、文件结构（无需理解，仅供备查）

```

style.css           样式

assets/             favicon、首页图
works/              作品图片（后台自动上传到这里）
data/works.json     作品数据
data/cv.json        履历数据
data/about.json     首页、About、Statement、Research、Contact 文字
admin/upload.html   后台
admin/admin.js      后台逻辑
```

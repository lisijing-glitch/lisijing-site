首次使用 · 设置步骤（点击展开/收起）
  1. 在 GitHub 打开 Settings → Developer settings → Personal access tokens → Fine-grained tokens，点 Generate new token。
  2. Repository access 选 Only select repositories，只勾选网站所在的这一个仓库。
  3. Permissions → Repository permissions → Contents 设为 Read and write，其余全部保持 No access。
  4. 设定有效期（例如 1 年），生成后把 github_pat_… 开头的字符串复制到下方「访问令牌」。
  5. 填好后点「保存设置」，看到右上角显示「已连接」即可开始使用。令牌只保存在这台设备的浏览器里。

只保存在本机浏览器，不会上传到任何地方。

安全提示：请只在自己的电脑/手机浏览器上打开本页并保存令牌；在公用设备上使用后请点「清除本机设置」。 后台页面本身是公开网址，但没有令牌的人打不开你的仓库，改不了任何内容。
